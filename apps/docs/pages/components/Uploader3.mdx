# Uploader3

An image upload React Component.

**Features**

- Supports cropping
- Supports uploading to web3 service providers, such as NFT.storage

## Import

```js copy
import { Uploader3 } from '@lxdao/uploader3';
```

## Props

| Prop         | Type                 | Required | Description                                            | Default                          |
| ------------ | -------------------- | -------- | ------------------------------------------------------ | -------------------------------- |
| accept       | `string`             | ❌       | image accept file type                                 | `['.png','.jpeg','.jpg','.gif']` |
| multiple     | `boolean`            | ❌       | multiple image upload                                  | `false`                          |
| api          | `string`             | ❌       | endpoint upload api url                                | `''`                             |
| connector    | `object`             | ❌       | create by [uploader3-connector](./uploader3-connector) | `null`                           |
| crop         | `object` / `boolean` | ❌       | crop config, set `false` disabled crop                 | `true`                           |
| onChange     | `function`           | ❌       | callback when files selected                           | `null`                           |
| onUpload     | `function`           | ❌       | callback when file uploading                           | `null`                           |
| onComplete   | `function`           | ❌       | callback when file uploaded                            | `null`                           |
| onCropEnd    | `function`           | ❌       | callback when crop end                                 | `null`                           |
| onCropCancel | `function`           | ❌       | callback when crop cancel                              | `null`                           |

<Callout type="info" emoji="ℹ️">
  `api` and `connector` are mutually exclusive, if both are provided, `api` will be used. must be provided one of them.
</Callout>
